# Ally Online Coach

Ally Online Coach is a comprehensive platform for personal trainers, offering a range of fitness coaching tools and services. This README provides an overview of the various components and technologies utilized in the development of this platform.

## Features

### Sanity CMS Integration
- Dynamic content sections such as blog and testimonials can be easily updated by the client.
- Leveraged Sanity CMS for efficient content management.

### Mobile-Responsive Design
- Ensures optimal user experience across devices.
- Utilizes Tailwind CSS for responsive design implementation.

### Dynamic Color Theme Changer
- Enhances visual aesthetics based on user interaction and navigation within the website.

### FitFolio
- Access to a diverse exercise database through an API.
- Optimized communication with FitnessAPI for efficient data retrieval.
- Utilizes React Router for seamless generation of dynamic pages.
- Planned future improvement: favoriting feature using LocalStorage.

### MotivateMe
- Personalized prompts for users from three AI personalities.
- AI-generated character images enhance user engagement.
- Demonstrates expertise in API integration and user experience design.

### ClientCheckin
- Web application for smooth check-ins and fitness tracking for trainer's clients.
- Utilizes MaterialUI's form options for seamless client interactions.
- Secure data storage and real-time updates using Firestore's authentication and database features.

## Technologies Used

- **Sanity CMS**: Content management system for dynamic content sections.
- **Tailwind CSS**: Framework for mobile-responsive design implementation.
- **Vite**: Tooling for fast and efficient web development.
- **React**: JavaScript library for building user interfaces.
- **FitFolio API**: Access to diverse exercise database.
- **Firebase Realtime Database**: Real-time data storage and updates.
- **OpenAI**: Conversational AI for personalized prompts.
- **MaterialUi**: UI framework for smooth client interactions.

## Installation and Usage

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Run the application using `npm start`.
5. Access the application through the provided local server address.
